Purejavascript

PureJavascript is a set of Javascript files that each provide useful functionality with absolutely no dependencies.

Download this project as a .zip file Download this project as a tar.gz file

Call( url, parameters, handler );

Call an API endpoint with the given parameters and call the handler with the AJAX responseText.

CloseModals();

Hides modals (sets display to "none" for DIVs that have a classname containing "modal") and also hides the modal background (a class with an id of "modal-bg".

GetFormValues( form )

Returns an object that contains as members the names/values of the INPUTs contained within the passed form.

GetSearchValues()

Returns an object that contains as members the names/values of any parameters passed in the search section of the current URL.

InsertFormValues( form, object )

Inserts the values of the members of the passed object into the INPUTs of the passed form.

Replace( string, object )

Replaces any substring of string that matches "%" + member + "%" with the value of the member in the object.

SetCookie( path, name, cvalue, exdays )

Sets a cookie.
